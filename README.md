# Python-WordleGame
Wordle is a word puzzle game where players try to guess a secret five-letter word within six attempts.

Learn how to play [Wordle](https://www.nytimes.com/games/wordle/index.html).  ([This video](https://www.youtube.com/watch?v=lv4Zg-209MY) may be helpful.)


**Examples:**
Scenario1: 
    User makes a sequence of guesses and wins the game in four steps:
Input Sequence:
    audio
    stern
    glyph
    audit
Watch for:
    Message indicating that the user wins the game.    


Scenario2:
    User enters words that have fewer than 5 letters
Input Sequence:
    audio
    safe
    step
    stop
    stomp
Watch For:
    Error message indicating required word length.
    Ability to continue game after entering 5 letter word
